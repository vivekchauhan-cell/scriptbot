Swipe File
P
PAS — Problem → Agitate → Solution
Lead with pain. Make it visceral. Position your product as relief. Best for cold audiences on Meta.
"Payday 10 din door? Wallet mein sirf ₹200? Hum samjhte hain. AbhiPay gives you salary now."
A
AIDA — Attention → Interest → Desire → Action
Classic funnel. Works across all formats and durations. Best for YouTube long-form.
"This app paid my salary 5 days early. → No interest. → 2M users. → Download free."
B
BAB — Before → After → Bridge
Show painful state → desired state → your product as the bridge. Strong for remarketing.
"Borrowing from friends → Now I get salary early → AbhiPay made that real."
C
CPB — Curiosity → Proof → Benefit
Open a loop, back with credibility, land the benefit. Best for app install campaigns.
"Why are 2M Indians using this app? RBI-registered, zero interest — get salary early."
Problem-first hooks outperform curiosity hooks by ~34% on Meta for fintech. For YouTube skippable ads, delay brand reveal to 8–10s to cut skip rates by 40%.

Script Blueprint
Proven structural frameworks by campaign objective
Performance
Remarketing
App Install
1
Hook 0–3s — Stop the Scroll
Lead with the pain point or bold claim. No brand logo yet. Visual-first: show the problem happening. Pattern interrupt is the only goal here.
2
Agitate 3–8s — Make It Personal
Amplify the problem emotionally. Use a relatable scenario. "Sound familiar?" moment. The audience must feel seen before they'll listen to your solution.
3
Solution Intro 8–15s — The Reveal
Introduce product as the natural answer. Show it in action — screen recording, demo, lifestyle. Keep it simple: demonstrate one core function only.
4
Proof 15–22s — Credibility Stack
1–2 proof points max: user count, rating, regulator badge, or media mention. More than 2 dilutes trust rather than building it.
5
CTA 22–30s — Single Clear Action
One action only. Urgency trigger if authentic. Reinforce the core benefit. "Download free. Get ₹5000 in 2 minutes." Zero confusion allowed.
1
Acknowledge 0–4s — They Know You
Skip the cold intro. Reference prior intent directly. "You explored us — let's talk about why 2M said yes." Warm audiences hate being treated as cold.
2
Objection Dissolution 4–12s
Name the #1 reason they didn't convert (trust, price, complexity). Lead with the objection, then dissolve it with a specific, credible proof point.
3
Urgency / Incentive 12–20s
Time-limited offer or consequence of inaction. "₹500 cashback this week only." The delay must feel genuinely costly without being manipulative.
4
Peer Proof 20–26s
Someone exactly like them made the choice. UGC video or testimonial from the same audience segment. Micro-relatability beats celebrity here.
5
Frictionless CTA 26–30s
"Takes 2 minutes. No documents. No calls." Emphasize ease over urgency. Remove every perceived barrier between them and the conversion action.
1
Lifestyle Hook 0–3s — Show the Outcome
Show someone's life AFTER the app. Not the product — the result. Real phone, real UI. Aspirational but undeniably relatable to your audience.
2
Screen Demo 3–12s — Make It Tangible
Live screen recording of the core action. Show 3 taps max to core value. Real data, no placeholder. Thumb-friendly UI clearly visible throughout.
3
Feature → Benefit Bridge 12–20s
Name 2–3 features. Immediately pivot to "what that means for you." Users buy outcomes not features — bridge every single time.
4
Install Proof 20–26s
Play Store / App Store rating + download count. "4.8 stars · 2M downloads." Platform trust signals work better here than brand trust alone.
5
Install CTA 26–30s
"Download free" + platform badge. Create separate iOS / Android cuts for better performance. Tie the final frame back to the lifestyle outcome shown in step 1.
Learning Loop
How ScriptBot improves with every campaign cycle
1
Generation — The Output Layer
Bot generates scripts using frameworks + learned patterns from your library. Each script gets a unique ID with metadata: hook type, campaign, tone, duration, platform.
2
Human Feedback — The Signal Layer
Your team rates each script (1–5 stars), tags quality issues ("weak CTA", "too wordy", "strong hook"), and marks it used or revised. This is the first feedback loop.
3
Campaign Performance — The Truth Layer
CTR, CVR, install rate, ROAS, VTR are linked back to script ID. High-performing scripts are auto-flagged as Gold templates. This is the strongest signal in the loop.
4
Pattern Extraction — The Intelligence Layer
Which hook types win for which verticals? What duration × platform × tone drives installs? Which words appear in every Gold script? Patterns become learned rules.
5
Prompt Refinement — The Loop Closes
Learned rules are injected back into generation context. "For fintech remarketing on Meta: lead with objection, use casual tone, avoid 'loan'" becomes a hard rule that improves every future script.
Signal strength: ROAS (strongest) → CVR → CTR → Human rating (weakest). Never over-index on taste — always tie feedback to downstream metrics.
Run each variant on ≥500 impressions before feeding performance data into the loop. Noisy signals mislead pattern extraction.
Every script saved to the Library with a performance data entry automatically updates your generation context. The bot gets smarter every cycle.
